Version 1.2.0 - AI Speaker Detection Release (2024-11-11)
----------------------------------------------------------
MAJOR NEW FEATURE: Local Voice-Based Speaker Detection!

🎤 Speaker Detection Features:
- AI-powered speaker identification using audio clustering
- 100% offline - no cloud services or accounts required
- Custom speaker name assignment (comma-separated)
- Configurable speaker count
- Speaker labels in all transcripts
- Full web UI integration

🐛 Bug Fixes:
- Fixed CLI flag mismatch (--two-list-quotes → --two-lists)
- Fixed speaker name mapping parameter bug
- Cleaned up UUID prefixes in file content
- Added persistent download links after container restart
- Fixed file display in web UI
- Changed port from 5000 to 5731

🔧 Technical Improvements:
- Embedded speaker recognition model (~87MB) in Docker image
- MFCC and spectral audio feature extraction
- AgglomerativeClustering for speaker grouping
- Intelligent fallback mechanisms
- Smart name mapping with user feedback

═══════════════════════════════════════════════════════════════
                     QUICK START
═══════════════════════════════════════════════════════════════

WEB INTERFACE (Easiest):
1. Install Docker Desktop
2. Run: ./transcribe-web.sh (Mac/Linux) or transcribe-web.bat (Windows)
3. Open: http://localhost:5731
4. Drag, drop, transcribe!

COMMAND LINE:
./transcribe.sh video.mp4 --speaker-diarization --speaker-names "Alice,Bob"
