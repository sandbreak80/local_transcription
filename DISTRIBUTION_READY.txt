🚀 QUICK START FOR NEW USERS

1. Install Docker Desktop
2. Run: ./transcribe.sh video.mp4 (Mac/Linux)
   Or:  transcribe.bat video.mp4 (Windows)
3. Done! Transcription saved next to video file
